# ProjetoFinalAttentus
Attentus
Descrição do Projeto

O Attentus é uma aplicação web desenvolvida com o objetivo de estimular capacidades cognitivas através de mini-jogos interativos. O projeto foi criado no âmbito da unidade curricular de Desenvolvimento Web e procura oferecer uma experiência simples, intuitiva e acessível.

A aplicação inclui:

Sistema de registo e autenticação de utilizadores;
Perfil de utilizador com progresso e estatísticas;
Sistema de XP, moedas e conquistas;
Loja de personalização de avatar;
Vários mini-jogos focados na atenção e concentração;
Sistema de níveis e dificuldades;
Painel administrativo para testes e desenvolvimento.
Tecnologias Utilizadas
HTML5
CSS3
JavaScript (ES6 Modules)
LocalStorage (armazenamento local dos dados)

Funcionalidades
Sistema de Utilizadores
Registo de novos utilizadores
Login através de utilizador ou email
Sessão guardada localmente
Logout
Perfil

Cada utilizador possui:

Nome de utilizador
XP acumulado
Nível
Saldo de moedas
Avatar personalizável
Estatísticas
Conquistas
Loja

A loja permite:

Comprar itens utilizando moedas
Equipar acessórios
Personalizar o avatar

Categorias disponíveis:

Chapéus
Óculos
Acessórios faciais
Troféus
Sistema de Recompensas

Ao concluir níveis ou jogos, o utilizador recebe:

XP
Moedas

Estas recompensas permitem:

Subir de nível
Desbloquear conquistas
Comprar itens na loja
Mini-Jogos
Jogo da Reação

Objetivo:

Clicar no carro apenas quando o semáforo ficar verde.

Competências trabalhadas:

Tempo de reação
Atenção visual
Repetir Sequência

Objetivo:

Memorizar e repetir a sequência de cores apresentada.

Competências trabalhadas:

Memória visual
Concentração
Jogo do Foco

Objetivo:

Encontrar todos os objetos escondidos numa imagem.

Competências trabalhadas:

Atenção ao detalhe
Observação
Jogo do Alvo

Objetivo:

Acertar no maior número possível de alvos antes do tempo terminar.

Competências trabalhadas:

Coordenação
Rapidez de resposta
Sistema de Dificuldades

Os jogos possuem três níveis de dificuldade:

Fácil
Médio
Difícil

Cada dificuldade contém vários níveis progressivos.

Sistema de Níveis

Os níveis são desbloqueados progressivamente.

Para avançar é necessário concluir o nível anterior.

O progresso é guardado automaticamente.

Conquistas

A aplicação inclui várias conquistas relacionadas com:

XP obtido
Níveis concluídos
Jogos realizados
Progressão geral

O progresso de cada conquista pode ser consultado no perfil.

Armazenamento de Dados

Os dados são guardados localmente através do LocalStorage, incluindo:

Utilizadores
Progresso
Estatísticas
Conquistas
Itens comprados
Configurações da aplicação

Considerações Finais

O Attentus foi desenvolvido com o objetivo de criar uma plataforma simples e divertida para estimular capacidades cognitivas através de mini-jogos interativos. O projeto permitiu aplicar conhecimentos de HTML, CSS e JavaScript, bem como conceitos de organização de código, persistência de dados e desenvolvimento de interfaces web responsivas.
